Linux

On debian, ubuntu and derivates, please use the package "openbve" from the repositorys. You can install via Softwarecenter or terminal. You can try using a ppa like this

On Archlinux and Manjaro,please install via pacman or octopi

For other distributions, look in their repositorys and software centers.

You can probably use the developer tools from windows in wine or mono.

Most Major Linux Distributions (Ubuntu, Fedora, Slackware, Etc.) Not compatible with ARM-based systems">


Mac

Download stable (7z)
Download developer edition (7z)

Requirements:Required Dependencies: OpenAL, SDL (Copy the SDL.framework directory to /Library/Frameworks), Mono (If you are given any choice, make sure that System.Windows.Forms and libgdiplus are installed.) XQuartz (For Lion, Mountain Lion, Mavericks & Yosemite Systems)

Because there isn't a real OpenBVE version for MacOS, you gave to run the Windows Version in Mono.

If you are having issues running the precompiled app, open terminal and do the following:

cd /path/where/openbve/is/installed
mono OpenBve.exe

To install add-ons on the pre-compiled app, right click (or control + click) and click on "Show package contents". Double click the contents folder, then double click the resources folder, then double click the userdata folder, and then double click the LegacyContent Folder. Place files on their respective folders.

OS X 10.5.8 or Later, Intel Macs. Not compatible with PowerPC Macs!
